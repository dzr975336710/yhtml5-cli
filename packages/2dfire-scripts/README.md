## How To Contribute

download git 

```
git clone git@git.2dfire-inc.com:static/2dfire-scripts.git
git checkout -b dev origin/dev

cd 2dfire-test
coding...
npm run test 
```

> npm info @2dfire/2dfire-scripts

npm publish 

```
npm login --registry http://10.1.6.28

cd packages/2dfire-scripts

npm publish
```
