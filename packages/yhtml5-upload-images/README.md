## yhtml5-update-images

An automated tool for compressing and optimizing JPEG or PNG images, 
and then upload to cdn return urls. 

> 压缩图片 => 上传到cdn => 返回url 

### old workflow
* 打开 https://tinypng.com/
* 上传图片
* 下载到文件夹
* 打开 上传图片 cdn 地址
* 上传图片
* 复制链接

### new workflow
* npm run upload
* return url
